Internet Technologies

What is Internet?

The Internet is a global network of interconnected computers, servers, and devices that communicate with each other to share information. It allows people around the world to access websites, send emails, watch videos, use social media, make online calls, and perform many other activities. When you use the Internet, data is sent and received through a system of networks using standard communication rules called protocols. In simple terms, the Internet connects billions of devices worldwide, enabling fast and easy communication and access to information from almost anywhere.

How the Internet works (Client-Server model, HTTP, DNS)

The Internet works mainly through the client-server model, where a client (such as a web browser on your computer or phone) sends a request for information, and a server (a computer that stores websites and data) responds by providing the requested information. When you enter a website address in your browser, the Domain Name System (DNS) first translates the domain name (such as google.com) into an IP address that identifies the server hosting the website. Once the IP address is found, the browser communicates with the server using HTTP (Hypertext Transfer Protocol) or its secure version, HTTPS. The server then sends the requested web page and related files back to the browser, which displays them for the user. This process happens within seconds, allowing users to access information and services on the Internet quickly and efficiently.

Main Services of the Internet:

  • World Wide Web (WWW) – Accessing websites and web pages.
  • Email – Sending and receiving electronic messages.
  • File Transfer (FTP) – Uploading and downloading files.
  • Remote Access – Accessing and controlling computers remotely.
  • Online Chat & Messaging – Instant communication between users.
  • Voice and Video Calls (VoIP) – Internet-based calling and video conferencing.
  • Streaming Services – Watching videos and listening to music online.
  • Social Networking – Connecting and sharing content with others.
  • Cloud Services – Online storage and computing resources.
  • Online Gaming – Playing games with other users over the Internet.

How the Internet is used in daily life

  • Communication through email, messaging apps, and video calls.
  • Accessing information and news from websites and search engines.
  • Online learning through educational platforms and courses.
  • Entertainment such as streaming movies, music, and online gaming.
  • Social networking and connecting with friends and family.
  • Online shopping and ordering products or services.
  • Internet banking and digital payments.
  • Remote work and collaboration using online tools.
  • Booking tickets, hotels, and transportation services.
  • Storing and sharing files using cloud services.

Role of Web Browsers

  • A web browser is software used to access and view websites on the Internet.
  • It sends requests to web servers and retrieves web pages.
  • It interprets and displays HTML, CSS, and JavaScript content.
  • It allows users to navigate between web pages using hyperlinks.
  • It provides features such as bookmarks, history, downloads, and tabbed browsing.
  • It helps users search for information, watch videos, and use online applications.
  • It ensures secure communication with websites through HTTPS.
